What is the Mixer Lookup Tool?

This tool allows users to view Mixer API information in a formatted way. This is very useful to see when accounts were created and all sorts of other public information. You can even use this tool to see who exactly is hosting you at any point in time.

How do I use it?

Type in a username below to look up their information!
